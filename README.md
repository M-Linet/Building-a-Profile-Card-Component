#  React Profile Card Component

A reusable React profile card component that displays user information including name, role, location, and bio. Built with Vite and styled with CSS. Demonstrates core React concepts: components, props, and reusability.

##  Assignment Requirements

This project fulfills the following requirements:
- ✅ Reusable `ProfileCard` component accepting: name, role, location, bio
- ✅ Follow button on each card
- ✅ Three profile cards with different information
- ✅ Kenyan representation (Nairobi-based developer)
- ✅ CSS styling with borders and proper spacing
- ✅ Clean, well-documented code structure
